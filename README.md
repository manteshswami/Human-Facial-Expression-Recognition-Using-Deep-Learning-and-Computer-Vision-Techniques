**😄 Human Facial Expression Recognition Using Deep Learning and Computer Vision Techniques**

**📌 Overview**

This project implements a Human Facial Expression Recognition (FER) system using a combination of Deep Learning and Computer Vision techniques. The system is capable of detecting and classifying human facial emotions from images in real time using Convolutional Neural Networks (CNNs) and OpenCV.
The goal is to build an end-to-end pipeline that:

* Detects human faces in an image or video frame
* Extracts facial features
* Classifies the expression into one of several emotional categories
